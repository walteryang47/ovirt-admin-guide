# 禁用自动负载均衡

在 oVirt 的集群调度策略中，可以通过禁用自动负载均衡来防止管理中心试图在优化过程中调整集群的平衡状态。

1. 以管理员身份登录到管理员门户。

2. 在头部栏中，点击**配置**。

3. 点击**集群策略**来查看集群策略标签。

4. 选择您的集群策略并点击**编辑**。另外，点击**复制**可以克隆并编辑默认的且锁定的集群。

5. 在 **Load Balancer** 的下拉列表中选择 *None*。

6. 点击**确定**。

7. 如果您已经通过**复制**创建了一个新的集群，请在**集群**标签中选择合适的集群，并点击**编辑**，在**集群策略**标签中选择新的策略。
